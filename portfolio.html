selfDrive project

this is a js and html project, originally following this tutorial, and later expanded through trial, error, and research.
you can save or delete the best-performing car and let mutations improve (or worsen) it — the mutation rate can be adjusted.
you can also change the number of input neurons and hidden-layer neurons, which resets the saved best car.
the number of lanes and traffic can be adjusted, as well as the number of self-driving cars.

after any change, simply refresh the page to start again.

and the fun part: using wasd keyboard inputs, you can race the best car yourself.

weather project

I made this project about a year ago while still in high school — it was mostly to get familiar with Swift and working with APIs, but it turned out quite well.
I built it using Xcode and the OpenWeather API, adding extra features like searching for cities and detecting the user’s location (if agreed).

//functionality:
earth globe icon: random weather of 20 cities around the world
city icon: weather in Debrecen
location icon: weather in user’s current location
magnifying glass icon: weather of searched city
+ includes forecasts as well

//extra info:
the scripts can be found in the weather directory, organized into separate subfolders.
to make it work, add a Secrets.plist file to the resources directory based on the provided Secrets-template.plist, and include your own API key from OpenWeather.
